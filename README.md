# Data Quality Analysis & Validation Using SQL

## Project Overview
This project focuses on analyzing structured relational data using SQL to identify data quality issues.
The project applies data profiling and validation checks across accuracy, completeness, consistency,
timeliness, and uniqueness.

## Tools Used
- SQL
- Relational Database (MySQL)

## Key Concepts
- Data Profiling
- Data Validation
- Data Quality Dimensions
- Relational Data
- Data Lineage
